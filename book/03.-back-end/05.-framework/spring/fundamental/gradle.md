# Gradle

## Gradle 설치

### Mac

```bash
# brew gradle
```

### Ubuntu

```bash
# 저장소 추가
$ add-apt-repository ppa:cwchien/gradle

# apt-get 업데이트
$ apt-get update

# Gradle 설치
$ apt-get install gradle
```

## Gradle 빌드

