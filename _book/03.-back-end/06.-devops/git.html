~/.ssh 경로에 다음 파일이 생성된다.

  • id_rsa
    • 개인키 (지정 경로 ~/.ssh 에 보관)
  • id_rsa.pub
    • 공개키 (복사해서 저장소에 넣음)

Parameter

  • -t
    • 암호화 타입
  • -b
    • 생성할 키의 비트수 (default: 2048)
  • -C
    • 주석입력

Revert

Syntax

$ git revert {COMMIT_NUMBER}

Merge Commit Revert

일반적인 리버트로는 불가능 하다.

-m 1 옵션을 넣어 줘야 한다.

$ git revert -m 1 {COMMIT_NUMBER}
