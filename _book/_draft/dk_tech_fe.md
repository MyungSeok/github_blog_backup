"자바스크립트에서 Hoisting 이란 무엇인가?"
함수 선언(리터럴 방식이 아닌)와 변수 선언은 런타임시 해당 유효범위의 맨 앞으로 옮겨지며 변수는 undefined를 기본 값으로 갖는다.
또한 이 때 유효범위내 같은 이름의 변수 선언과 함수 선언이 있을 경우 변수의 초기값이 있다면 변수가 우선순위를 가지며 그렇지 않을 경우 함수가 우선순위를 갖는다.

"자바스크립트에서 == 와 === 의 가장 큰 차이는 무엇인가?"
== 는 먼저 데이터 타입이 같도록 형변환을 거친 후 값을 비교한다.

Array.prototype.filter() - 인자로 넘겨진 callback의 조건에 맞는 값들로 구성된 새로운 array를 리턴해준다.
