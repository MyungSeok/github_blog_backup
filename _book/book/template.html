Description

  • 이건 이것이고 저건 저것이다.

Syntax

public class Foo {
  public static void main (String[] args) {
    System.out.println("Hello World");
  }
}

Parameters

  • Foo
    • 클래스명
  • args
    • 매개변수

  • 반환값 없음

내용 정리

Example

Case 1

(function (param) {
  console.log(param || 'Hello World');
}());
